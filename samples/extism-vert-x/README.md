### Vert-x project

- Go to https://start.vertx.io/
- Choose `Java` for **Language**
- Choose `Maven` for **Build**
- Add the Vert.x Web dependency
- Click on the "Generate project" button (it will start the download of the `starter.zip` file)
- Unzip it
